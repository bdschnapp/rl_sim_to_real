TRAILER PLANNING PACKAGES ANALYSIS

ARCHITECTURE CONTEXT
--------------------
The repo uses a scenario-switching approach: autoware_scenario_selector routes to either the
standard pipeline (LANEDRIVING) or the dedicated trailer pipeline (TRUCKTRAILER). The trailer
planner + bridge replace the motion planning stage — they don't need the entire standard
pipeline to be rewritten. Updates are about what the standard pipeline touches when not in
trailer mode vs. what needs trailer awareness.
